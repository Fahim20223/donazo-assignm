# Donezo - Project Management Dashboard

A modern, responsive project management dashboard built with React, Tailwind CSS, and DaisyUI.

## Features

- 📊 Project analytics with interactive charts
- 👥 Team collaboration tracking
- ⏱️ Built-in time tracker
- 🌓 Dark mode support
- 📱 Fully responsive design
- 🎨 Modern UI with Tailwind CSS & DaisyUI

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- DaisyUI
- Material Symbols Icons
